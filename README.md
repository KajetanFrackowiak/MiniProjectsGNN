# MiniProjectsGNN

A collection of mini research projects focused on Graph Neural Networks (GNNs).

## Projects

- **[GAT (Graph Attention Networks)](./GAT/)** - Implementation of Graph Attention Networks using PyTorch Geometric
  - Datasets: Cora, CiteSeer, PubMed, PPI
  - Transductive and inductive learning tasks

## Documentation

Each project has its own README with documentation, setup instructions, and benchmarks.

## General

### Setup
1. Clone and navigate to the repository
2. Install pre-commit hooks:
   ```bash
   pre-commit install
   ```
3. Each project has its own setup instructions and prerequisites - see individual README files
